## 2. 源码结构

### 2.1 前端文件

| 文件 | 行数 | 用途 |
|------|------|------|
| `app.js` | 472 | 前端逻辑（WebView 桥接、模块管理、ROOT 状态显示） |
| `index.html` | — | UI 结构（概览、模块页面、底部导航） |
| `styles.css` | — | 共享样式（玻璃拟态、动画） |
| `logo.png` | — | 应用图标 |

### 2.2 Kotlin 后端文件

| 文件 | 用途 |
|------|------|
| `RootManager.kt` | ROOT 状态检测、Zygisk/Ramdisk 信息显示 |
| `ModuleManager.kt` | 模块列表、启用/禁用/删除操作 |
| `ModuleInstallManager.kt` | 从 ZIP 文件安装模块 |
| `RootShell.kt` | Shell 命令执行 |
| `CuSO4Bridge.kt` | JavaScript ↔ Kotlin 桥接 |
| `MainActivity.kt` | 主 Activity |

**说明**：BackupManager.kt、AccountManager.kt、MarketManager.kt **未包含**，因为这些功能不属于开源版本。

## 5. 构建方法

```bash
# 1. 克隆仓库
git clone https://github.com/CuSO4-X/CuSO4-RootManager.git
cd CuSO4-RootManager

# 2. 构建 APK
./gradlew assembleDebug

# 3. 验证 SHA256 哈希值
sha256sum app/build/outputs/apk/debug/app-debug.apk
# 预期：8CEEBD9BD2CC719EE6BDE2B5ABC27A73DAC27253DF365B47F4246D4715D51509
```

## 6. 总结

**CuSO4 Open 是一个简洁、完全开源的 ROOT 管理器。**

- **开源**：所有源码均在 GitHub 上
- **无网络**：零 HTTP 请求，无数据收集
